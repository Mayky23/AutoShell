# AutoShell - Generador de Shells Reversas

AutoShell es una herramienta para generar shells reversas en distintos lenguajes de programación, adaptadas para Windows, Linux y macOS. Su objetivo es facilitar el aprendizaje de ciberseguridad en entornos controlados.

## 🚀 Características  
✅ **Genera shells reversas en Python, PHP, Node.js, Bash y PowerShell**.  
✅ **Permite seleccionar el sistema operativo de la víctima**.  
✅ **Crea el script del cliente**. 
✅ **Permite la ejecución remota de comandos desde la herramienta**.
✅ **Shells completamente interactivas**.  
✅ **Compatible con Linux, Windows y macOS**.  

---  

## 📌 Instalación y Uso  

### 🔹 Instalación  
Clona este repositorio en tu máquina:  
```bash
git clone https://github.com/tuusuario/autoshell.git  
cd autoshell  
```

### 🔹 Ejecución  
```bash
python3 AutoShell.py  
```

### 🔹 Configuración  
1️⃣ Selecciona el sistema operativo de la víctima.  
2️⃣ Selecciona el lenguaje de la shell.  
3️⃣ Introduce la IP atacante y el puerto.  
4️⃣ Elige la ruta de guardado.
